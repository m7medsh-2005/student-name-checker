# Student Name Checker

A simple student name checker project built using HTML, CSS, and JavaScript.

The user enters a student name, and the app checks if the name exists inside a predefined list of students.

## Features

- Input for student name
- Button to check the name
- Shows if the student is found
- Shows if the student is not found
- Uses array, loop, condition, and string methods

## Technologies Used

- HTML
- CSS
- JavaScript
